# (PC PARTS)MVVM Activity 

IPT activity for MVVM (Model View ViewModel).
A simple Android application demonstrating the **MVVM (Model-View-ViewModel)** architecture pattern in Java. This app displays a list of computer parts with specifications and images using a `RecyclerView`.
